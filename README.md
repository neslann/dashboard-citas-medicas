
# Dashboard de Gestión de Citas Médicas

Proyecto desarrollado para el Laboratorio N.° 13 del curso Business Intelligence and Big Data.

## Descripción

Este proyecto simula un conjunto de datos de 5000 citas médicas de un hospital de Lima Norte.

El objetivo es analizar el comportamiento de las citas médicas, identificar las especialidades con mayor demanda, detectar cancelaciones y ausencias, y visualizar los resultados mediante un dashboard interactivo.

## Herramientas utilizadas

- Python
- Pandas
- Plotly
- Streamlit
- Google Colab
- GitHub
- Streamlit Community Cloud
